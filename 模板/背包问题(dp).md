以下问题中m为可选物品数量，n为总可用钱数，f[i][j]表示用j块钱在前i件物品中最多可以选到多少钱的物品。w[i]为i号物品的费用（体积），v[i]为i号物品的价值

01背包，无优化
```cpp
for(int i = 1; i <= m; i++)
		for (int j = 0; j <= n; j++) {
			f[i][j] = f[i - 1][j];
			if(j >= w[i])
				f[i][j] = max(f[i - 1][j], v[i] + f[i - 1][j - w[i]]);
		}
cout << f[m][n];
```
